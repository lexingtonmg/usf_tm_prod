Theatre Manager Integrated Web Module

This page is explains the structure of the Theatre Manager web module.

All interaction with the Theatre Manager web module is HTML pages using FORMS and either POST or GET methods. They are relatively simple to create with any editor and because they are HTML only, you can integrate any of the pages into your existing web site and colour them or add graphics in a number of ways.

There are 4 frames in this example:

  • The left frame is called F_HTML_FIREWALLtmnavSide.html
  • The top frame is called tmnavTop.html
  • This frame is called tmnavMain.html
  • The bottom frame is called tmnavBottom.html

All pages in the demo are sent back to the frame that they came from. In this case, the replies from the server will be in the frame called 'main' and will replace this window.

To begin, click the Online Sales button on the side frame.
